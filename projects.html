Smart Home Thermostat

I built this one several years ago and it is still in use today. This was somewhat just for my own entertainment, but also to do a long-term experiment to develop a code base for general building automation. It has proven to perform quite well.

This HVAC controller accounts for unbalanced duct work and integrates a previously separate system for a heated floor in an addition. Based on multiple temperature sensors throughout the house, it circulates air between cycles if some rooms are noticeably warmer or cooler than others. It also targets the average of these temperatures when calling for heat or A/C. These two features combined ensure a more consistent temperature throughout the house and more efficient use of the mechanical system.

This was an entertaining project combining physical controls and a web UI. It has gone through several iterations and I hope to further improve it over time, with the goal of making it more modular and scalable.

PLC Code Generation

This is a more recent project that is functional, but still is very much in development. The program structure for most industrial control applications should follow a relatively consistent pattern. Once this structure is understood, setting up new programs is fairly easy but can be time consuming.

Over the years in this field I have taken on many new tasks such as project management, estimating project budgets, and other administrative tasks. As I lose more and more time to these tasks I am left with less time to write code and this project is an effort to reduce the amount of time needed to develop new programs.

The current iteration of this is based on standardized data structures and functions that I have built up over the years. It also is based on simple text files as the input data, which requires very strict use of device nomenclature. The next iteration will support the use of a formatted spreadsheet with drop-down lists that will allow for more flexibility in naming conventions. These should still follow a consistent standard, but will support a wider range of applications. The final goal would be to allow users to import their own structures and functions.
